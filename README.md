# 🤖 AI Personal Assistant (Vertex AI + ADK)

An AI-powered personal assistant built using Google Cloud Vertex AI via the Agent Development Kit (ADK), capable of understanding and responding to natural language queries through an interactive web interface.

---

## 🚀 Features

* 🧠 Natural Language Understanding
* 💬 Context-aware AI responses
* 🌐 Web-based interaction using ADK
* ⚡ Powered by Vertex AI models
* ☁️ Cloud-integrated AI system

---

## ⚙️ How to Run the Project

### 1. Navigate to project directory

```
cd ai-agents-adk/personal_assistant
```

### 2. Start the agent using ADK

```
adk web
```

### 3. Open in browser

After running, open the local URL shown in terminal (usually):

```
http://localhost:8000
```

## ⚠️ Note on Setup

This project uses Google Cloud Vertex AI and requires proper cloud configuration.

To run this project, you will need:

* A Google Cloud account
* Vertex AI API enabled
* Proper authentication setup
* Environment variables configured in `.env`

Due to these dependencies, direct execution may require setup on Google Cloud.
